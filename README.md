1.	Install XAMPP - https://www.apachefriends.org/download.html
2.	Clone GitHub project into XAMPP Site folder - https://github.com/drewhy98/referralportal.git 
3.	In XAMPP, click start Apache and mySQL
4.	Run the queries in the init.sql file to create the database and tables, using php myadmin -http://localhost/phpmyadmin/index.php?lang=en_GB
5.	Portal will now be reachable under local host - http://localhost/referralportal/site/
